wkSelectInput 使用说明文档

基于jQuery的插件

初始化方法：

$('#abc').initWkSelect(option);

* @param data * @param emptyTip * @param placeholder * @param width * @param height * @param type 1-选择后直接填充在输入框；2-选择后填充成标签形式 * @param inputCb * @param readonly 针对type=1的属性，默认选择后可以修改，readonly=“readonly”时不可修改

更新下拉列表数据方法：

$('#abc').updateWkSelectList(data,emptyTip,cb)

* @param data 下拉里列表数据源 数组，格式：[{name:'',id:'',key:''}] * @param emptyTip 下拉列表为空提示（默认值：暂无数据）

输入框设置默认选中值：

setDefaultInput(data,type,readonly)

* @param data 填充的数据，格式{name:'',id:'',key:''} * @param type 1-直接显示在input框；2-在input中用标签展示 * @param readonly 针对type=1的属性，默认可以修改，readonly=“readonly”时不可修改

获取已选择的数据的name值：

getWkVal()

获取已选择的数据的Value值：

getWkId()

获取已选择的数据的key值：

getWkKey()

是否选择了下拉列表；true-已选择；false-未选择；：

isChooseWkItem

输入框点击事件：

wkInputClick(cb)

@param cb 回调

输入框keyup事件：

wkInputKeyup(cb)

输入框点击和keyup综合事件：

wkInputClickMul(cb)

选择下拉框点击事件：

wkSelectClick(cb)

type==2时选中的标签关闭取消点击事件：

wkLabelClose(cb)
